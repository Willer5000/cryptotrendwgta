Estrategia de Trading

Esta estrategia combina múltiples indicadores técnicos para identificar oportunidades de trading:

  • Líneas de tendencia: Confirmadas con EMAs (9 y 20 períodos)
  • Divergencias: Detectadas con RSI (14 períodos)
  • Soportes/Resistencias: Calculados con ventana de 20 velas
  • Confirmación: Volumen significativo y ADX > 25

Recomendaciones:

  • Operar en timeframe de 4h para mejores resultados
  • Usar gestión de riesgo (máximo 1-2% por operación)
  • Confirmar señales con múltiples timeframes

Sobre el Indicador

Este indicador analiza las 150 principales criptomonedas cada 15 minutos usando la API pública de KuCoin. Las señales se generan basadas en:

  • Convergencia de indicadores técnicos
  • Volumen significativo
  • Divergencias RSI
  • Niveles clave de soporte/resistencia

Las operaciones se calculan con riesgo:beneficio mínimo de 1:3 usando ATR para determinar SL y TP.
